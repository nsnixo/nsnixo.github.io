To-Do List

Task Manager App

A simple to-do list app where you can add, view, and delete tasks. I learned about lists and loops with this project.

Python Lists Loops
todo_list.py Copy 
# My to-do list app
tasks = []

def show_tasks():
    if not tasks:
        print("No tasks yet!")
    else:
        for i, task in enumerate(tasks, 1):
            print(f"{i}. {task}")

while True:
    print("\n1. Add task")
    print("2. View tasks")
    print("3. Delete task")
    print("4. Quit")
    
    choice = input("Choose: ")
    
    if choice == '1':
        task = input("Enter task: ")
        tasks.append(task)
        print("Task added!")
    
    elif choice == '2':
        show_tasks()

Number Game

Guess the Number

A fun number guessing game. The computer picks a random number and you try to guess it. I learned about random numbers and while loops.

Python Random Loops
guess_game.py Copy 
import random

# Number guessing game
secret = random.randint(1, 100)
attempts = 0
guessed = False

print("I'm thinking of a number 1-100!")

while not guessed:
    guess = int(input("Your guess: "))
    attempts += 1
    
    if guess < secret:
        print("Too low!")
    elif guess > secret:
        print("Too high!")
    else:
        print(f"Yes! It's {secret}!")
        print(f"You got it in {attempts} tries!")
        guessed = True

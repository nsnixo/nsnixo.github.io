E-commerce Price Tracker

A Python script using BeautifulSoup and Requests to track prices across multiple e-commerce sites and send email alerts.

Python BeautifulSoup Requests SMTP Pandas
scraper.py Copy Code 
import requests
from bs4 import BeautifulSoup
import smtplib

def check_price(url, target_price):
    # Get page content
    headers = {'User-Agent': 'Mozilla/5.0'}
    page = requests.get(url, headers=headers)
    
    # Parse HTML
    soup = BeautifulSoup(page.content, 'html.parser')
    
    # Extract price
    price_element = soup.find('span', {'class': 'price'})
    current_price = float(price_element.text.replace('$', ''))
    
    if current_price < target_price:
        send_alert(url, current_price)
    
    return current_price

Sales Dashboard

Interactive dashboard analyzing sales data using Pandas, Matplotlib, and Plotly with automated reporting features.

import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

# Load data
df = pd.read_csv('sales_data.csv')

# Data cleaning
df['Date'] = pd.to_datetime(df['Date'])
df['Month'] = df['Date'].dt.month

# Monthly sales analysis
monthly_sales = df.groupby('Month')['Revenue'].sum()

# Visualization
plt.figure(figsize=(10, 6))
sns.lineplot(x=monthly_sales.index, y=monthly_sales.values)
plt.title('Monthly Revenue Trend')
plt.xlabel('Month')
plt.ylabel('Revenue ($)')
plt.savefig('monthly_trend.png')

File Organizer Bot

Automated file organization system that categorizes files by type, date, and size with intelligent naming conventions.

Python OS Module Pathlib Watchdog Automation
organizer.py Copy Code 
import os
import shutil
from pathlib import Path
from watchdog.observers import Observer

class FileOrganizer:
    def __init__(self, watch_folder):
        self.watch_folder = Path(watch_folder)
        self.file_types = {
            '.pdf': 'Documents',
            '.py': 'Python',
            '.jpg': 'Images',
            '.csv': 'Data'
        }
    
    def organize_existing(self):
        for file in self.watch_folder.iterdir():
            if file.is_file():
                self.move_file(file)
